# Бот для учебных работ

Telegram-бот для приёма заказов на написание учебных работ — дипломы, диссертации, рефераты, контрольные и другие учебные работы.

## О проекте

Бот общается со студентами от имени специалиста по имени Инга. Он собирает все необходимые данные о заказе через удобный диалог и отправляет готовый бриф владельцу сервиса в Telegram и на email.

Два режима подачи заявки:
- 📝 **Обычная анкета** — клиент отвечает текстом
- 🎤 **Быстрая заявка** — клиент отвечает голосом, бот расшифровывает

## Технологии

- Python 3.11+
- aiogram 3 — фреймворк для Telegram-ботов
- SQLite + aiosqlite — хранение заявок
- aiosmtplib — отправка email-уведомлений
- openai (Whisper) — расшифровка голосовых сообщений

## Запуск

1. Скопируй `.env.example` в `.env` и заполни значениями
2. Установи зависимости: `pip install -r requirements.txt`
3. Запусти бота: `python main.py`

_Полная инструкция появится по мере разработки_
