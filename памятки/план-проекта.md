# План проекта: Бот для учебных работ

## Что за проект

Telegram-бот **Инга** — автоматический диспетчер сервиса по написанию учебных работ на заказ. Бот общается со студентами, собирает данные о заказе и отправляет готовый бриф владельцу в Telegram и на email.

Два режима: обычная анкета (текст) и быстрая заявка (голос с расшифровкой).

## Что делать дальше — по шагам

### Шаг 1 — Подготовка окружения
- Установить Python 3.11+
- Создать виртуальное окружение: `python -m venv venv`
- Создать файл `requirements.txt` с зависимостями (aiogram, aiosqlite, aiosmtplib, openai, python-dotenv)
- Установить зависимости: `pip install -r requirements.txt`
- Скопировать `.env.example` в `.env` и заполнить реальными значениями

### Шаг 2 — Получить токен бота
- Написать @BotFather в Telegram → создать нового бота → получить BOT_TOKEN
- Узнать свой Telegram ID через @userinfobot → записать в OWNER_TELEGRAM_ID
- Подготовить почту для отправки писем (или использовать inga17@mail.ru)
- Получить ключ OpenAI для Whisper (openai.com → API keys)

### Шаг 3 — Разработка бота
Передать разработчику три документа:
1. `промпт_бот_учебные_работы_v4.md` — вся логика диалога и сценарии
2. `ТЗ_бот_учебные_работы.md` — техническое задание
3. `.env.example` — список нужных переменных

Что разработчик должен реализовать:
- [ ] Точка входа `main.py` — запуск бота
- [ ] Приветствие с кнопками «📝 Обычная анкета» / «🎤 Быстрая заявка»
- [ ] Обычная анкета — FSM (машина состояний), 9 вопросов по одному
- [ ] Быстрая заявка — те же вопросы, голос → Whisper → показать расшифровку → подтвердить
- [ ] Экран подтверждения с кнопками «✅ Подтверждаю» / «✏️ Редактировать»
- [ ] Отправка брифа владельцу в Telegram (личное сообщение)
- [ ] Отправка брифа на email inga17@mail.ru
- [ ] Пересылка прикреплённых файлов и голосовых сообщений владельцу
- [ ] Фильтр физмат-специальностей
