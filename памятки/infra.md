# Инфраструктура проекта

## Основное

- **Проект:** Бот для учебных работ
- **Тип:** Telegram-бот
- **Технологии:** Python 3.11+, aiogram 3, SQLite, aiosqlite, aiosmtplib, openai
- **Создан:** 2026-05-02

## Окружение

Все переменные окружения хранятся в файле `.env` (см. `.env.example`):

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен Telegram-бота от @BotFather |
| `OWNER_TELEGRAM_ID` | Telegram ID владельца для получения брифов |
| `OWNER_EMAIL` | Email владельца (inga17@mail.ru) |
| `EMAIL_LOGIN` | Почта, с которой бот отправляет письма |
| `EMAIL_PASSWORD` | Пароль от этой почты |
| `OPENAI_API_KEY` | Ключ OpenAI для Whisper (расшифровка голоса) |

## Зависимости

Установить командой: `pip install -r requirements.txt`

| Библиотека | Назначение |
|---|---|
| aiogram 3 | Telegram-бот (асинхронный фреймворк) |
| aiosqlite | Асинхронная работа с SQLite |
| aiosmtplib | Отправка email |
| python-dotenv | Загрузка переменных из .env |
| openai | Whisper API для расшифровки голоса |

## Команды

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск бота
python main.py
```
