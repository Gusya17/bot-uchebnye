# Бот-диспетчер учебных работ «Инга»

## What This Is

Telegram-бот для автоматического приёма заказов на написание учебных работ. Бот выступает в роли вежливого администратора «Инги» — встречает студента, ведёт структурированный диалог, собирает все данные заказа и отправляет готовый бриф владельцу в Telegram и на email. Владелец получает структурированный заказ и уже без лишних переспросов связывается с клиентом лично.

## Core Value

Каждый новый заказ автоматически приходит к специалисту в виде готового брифа — без ручного сбора данных, без потери деталей.

## Context

- **Владелец:** специалист по гуманитарным учебным работам (физмат/технические не берёт)
- **Клиенты:** студенты российских вузов, нуждающиеся в написании работ
- **Специализация:** юриспруденция, госуправление, экономика, психология, педагогика, культурология, вокальное искусство и смежные направления
- **Email владельца:** inga17@mail.ru
- **Время ответа владельца:** в течение 2 часов по МСК

### Active

**Сбор заявки:**
- [ ] Двухрежимный диалог: «Обычная анкета» (текст) и «Быстрая заявка» (голос + Whisper)
- [ ] Приветствие от имени «Инги» при любом первом сообщении
- [ ] Меню из 9 типов работ (диссертация, диплом, курсовая, контрольная, реферат, доклад, отчёт, доклад+презентация, другое)
- [ ] Сбор обязательных полей: имя, телефон, тип работы, тема, срок сдачи
- [ ] Сбор необязательных полей: учебное заведение, факультет, уровень обучения, методические требования (файл/текст), особые пожелания (текст/голос)
- [ ] Проверка телефона — обязательное поле, запрашивать до получения
- [ ] Проверка на физмат — вежливый отказ по специализации/теме
- [ ] Подтверждение анкеты клиентом перед отправкой (кнопки: Подтвердить / Редактировать)
- [ ] Расшифровка голосовых через Whisper API (для режима «Быстрая заявка» и особых пожеланий)

**Отправка брифа:**
- [ ] Отправка брифа владельцу в Telegram (личным сообщением)
- [ ] Отправка брифа на email inga17@mail.ru
- [ ] Пересылка прикреплённых файлов и голосовых вместе с брифом
- [ ] Подтверждение клиенту: «Заявка принята, свяжемся в течение 2 часов»

**Хранение и инфраструктура:**
- [ ] Сохранение заявок в SQLite (история)
- [ ] Конфигурация через .env (токены, email, Telegram owner ID)
- [ ] Обработка повторных обращений (новый заказ или вопрос по текущему)

### Out of Scope

- Расчёт стоимости — называет только владелец после изучения брифа
- Физмат/технические работы — явный отказ
- Личный кабинет или веб-интерфейс — только Telegram
- Приём оплаты — вне scope

## Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Python + aiogram 3 | Асинхронный стек, зрелый фреймворк для Telegram-ботов | Принято |
| SQLite + aiosqlite | Простота, нет нужды в отдельном сервере БД | Принято |
| Whisper API (openai) | Распознавание голоса для режима «Быстрая заявка» | Принято |
| aiosmtplib | Асинхронная отправка email | Принято |
| Имя бота — «Инга» | Создаёт образ живого администратора, вызывает доверие | Принято |
| Два режима подачи | Снижает барьер — студент выбирает удобный формат | Принято |

## Tech Stack

- **Runtime:** Python 3.11+
- **Bot framework:** aiogram 3
- **Database:** SQLite + aiosqlite
- **Email:** aiosmtplib
- **Voice recognition:** openai (Whisper API)
- **Config:** python-dotenv (.env)
