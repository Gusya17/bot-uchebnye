# Сценарий приёма заявки на учебную работу

```mermaid
flowchart TD
    START([Пользователь]) -->|/order| S1

    CANCEL([/cancel]) -->|в любой момент| MENU([Главное меню])

    S1["🎓 Приветствие\n«Привет! Я помогу оформить заявку на учебную работу.\nС нами — от заявки до защиты 🎓»"]
    S1 --> S1b

    S1b["Ваша работа — гуманитарная дисциплина?"]
    S1b --> B0{Направление}
    B0 -->|Да, гуманитарная| S2
    B0 -->|Технические науки физмат| REJECT

    REJECT["😔 К сожалению, технические и точные науки —\nне моя специализация.\nЯ работаю только с гуманитарными направлениями."]
    REJECT --> END_REJ([Сценарий завершён])

    S2["Шаг 1 из 5\nВыбор типа работы"]
    S2 --> B1{Тип работы}
    B1 -->|Другое| S3
    B1 -->|Диплом магистра| S3
    B1 -->|Диплом бакалавра| S3
    B1 -->|Курсовая| S3
    B1 -->|Реферат| S3
    B1 -->|Контрольная| S3

    S3["Шаг 2 из 5\nВведите тему работы\n(свободный текст)"]
    S3 -->|пользователь вводит тему| S4

    S4["Шаг 3 из 5\nВыбор срока сдачи"]
    S4 --> B2{Срок}
    B2 -->|до 3 дней| S5
    B2 -->|до 7 дней| S5
    B2 -->|до 14 дней| S5
    B2 -->|до 30 дней| S5

    S5["Шаг 4 из 5\nВведите контакт для связи\n(телефон или Telegram)"]
    S5 -->|пользователь вводит контакт| S6

    S6["Шаг 5 из 5\nПодтверждение заявки\n— Тип: …\n— Тема: …\n— Срок: …\n— Контакт: …"]
    S6 --> B3{Подтвердить?}
    B3 -->|✅ Подтвердить| S7
    B3 -->|❌ Отменить| MENU

    S7["✅ Заявка принята!\nСпасибо, мы скоро свяжемся"]
    S7 --> END([Конец сценария])

    S1 -.->|/cancel| MENU
    S1b -.->|/cancel| MENU
    S2 -.->|/cancel| MENU
    S3 -.->|/cancel| MENU
    S4 -.->|/cancel| MENU
    S5 -.->|/cancel| MENU
    S6 -.->|/cancel| MENU

    style S7 fill:#d4edda,stroke:#28a745,color:#155724
    style END fill:#d4edda,stroke:#28a745,color:#155724
    style REJECT fill:#f8d7da,stroke:#dc3545,color:#721c24
    style END_REJ fill:#f8d7da,stroke:#dc3545,color:#721c24
    style MENU fill:#fff3cd,stroke:#ffc107,color:#856404
    style CANCEL fill:#f8d7da,stroke:#dc3545,color:#721c24
```

## Описание шагов

| Шаг | Экран | Тип ввода | FSM-состояние |
|-----|-------|-----------|---------------|
| 0 | `/order` → приветствие | — | `OrderStates.checking_direction` |
| 0.5 | Гуманитарная / физмат? | Inline-кнопки | `OrderStates.checking_direction` |
| — | Физмат → вежливый отказ | — | сброс FSM |
| 1 | Тип работы | Inline-кнопки | `OrderStates.choosing_type` |
| 2 | Тема работы | Свободный текст | `OrderStates.entering_topic` |
| 3 | Срок сдачи | Inline-кнопки | `OrderStates.choosing_deadline` |
| 4 | Контакт для связи | Свободный текст | `OrderStates.entering_contact` |
| 5 | Подтверждение | Inline-кнопки | `OrderStates.confirming` |
| — | `/cancel` → главное меню | Команда | сброс FSM |

## Данные, собираемые в заявке

- **Направление** — гуманитарная (физмат отсекается на старте)
- **Тип работы** — Контрольная / Реферат / Курсовая / Диплом бакалавра / Диплом магистра / Другое
- **Тема** — произвольный текст от пользователя
- **Срок** — до 3 / 7 / 14 / 30 дней
- **Контакт** — телефон или @username в Telegram
- **Telegram ID и username** — берутся автоматически из объекта `message.from_user`
